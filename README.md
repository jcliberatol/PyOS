PyOS
====

A distributed python OS Emulator for adhoc networks



## Usage : 
```
Server.py start     SERVERIP DeviceName
          addnode   node.info
          allprocs  
          procs     nodeID
          exec      nodeID    command
          execall   command
```
## Set up a node

in the node

```

node.py start node.info
node.py printNode
node.py command
```
